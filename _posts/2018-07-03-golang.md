Brief setup steps of Golang in Windows computer

1. Go to your Environment Variables setting and set the following `System Variables`:

    | Variable    | Value              | 
    |:-----------:|:------------------:|
    | GOROOT      | C:\go              |
    | GOPATH      | C:\goWorkspace     |
    | Path        | C:\go\bin          |
    | Path        | C:\goWorkspace\bin |
